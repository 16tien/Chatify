🌟 Giới thiệu
Chatify là ứng dụng chat realtime, với các tính năng:
- Chat 1-1 realtime
- Thông báo push khi có tin nhắn mới
- Video call realtime ZegoCloud 
- Lưu lịch sử chat bằng Firebase Firestore

## 🚀 Tính năng
- Đăng nhập / đăng ký bằng email/password
- Chat realtime với Firebase Firestore
- Push notification khi có tin nhắn mới
- Gọi video bằng ZegoCloud
- Tìm kiếm và thêm bạn bè

## 🛠 Công nghệ sử dụng
- **Flutter** - Frontend
- **Firebase** - Authentication, Firestore, Cloud Messaging (FCM)
- **Provider** - Quản lý trạng thái
- ZegoCloud  - Video Call
